# Advanced Number Guessing Game in Python

An advanced command-line Number Guessing Game built using Python. The game includes difficulty levels, score tracking, best score record, input validation, smart hints, timer, statistics, and replay support.

---

## Features

- Play Again Option
- Difficulty Levels (Easy, Medium, Hard)
- Score System
- Best Score Record
- Input Validation
- Out of Range Check
- Smart Hints
- Attempt Counter
- Correct Number Reveal
- Timer
- Game Statistics

---

## Difficulty Levels

| Level | Number Range | Attempts |
|------|------:|------:|
| Easy | 1 – 10 | 5 |
| Medium | 1 – 50 | 7 |
| Hard | 1 – 100 | 10 |

---

## Score System

| Attempts Used | Score |
|-------------:|------:|
| 1 | 100 |
| 2 | 90 |
| 3 | 80 |
| 4 | 70 |
| 5 | 60 |
| ... | Minimum 10 Points |

---

# Smart Hints

The game provides helpful hints based on how close your guess is:

- Extremely Close!
- Very Close!
- Getting Closer!
- Far Away!

It also tells you whether to choose a smaller or larger number.
